Latest Project

Lately I built full-stack web app that allows users to create and manage soccer leagues. Features include full authentication with password recovery, authorization (role based access), created league collaboration, fixture creation and result and fixture updates. This project is available on GitHub and was built with React.js for dynamic rendering, CSS for styling, Java Spring Boot framework for a RESTFul API and MySQL as the database.
